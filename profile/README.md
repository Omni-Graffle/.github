# Omni Graffle – Professional Diagramming & Vector Illustration Tool

## About Omni Graffle
Omni Graffle is a professional-grade diagramming and vector illustration application designed for precision technical drawing, wireframing, and graphic design.  
It provides advanced tools for creating flowcharts, UI mockups, network diagrams, and detailed layouts. Omni Graffle offers robust stencil libraries, multi-page document support, and granular typography controls, making it an essential tool for designers, developers, and technical illustrators.  
Omni Graffle delivers professional functionality with a streamlined interface, ideal for both complex projects and rapid prototyping.

## Getting Started

1.  Access the Omni Graffle toolkit and extract the package.
2.  Launch Omni Graffle directly from the folder (no complex setup required).
3.  Log in via phone number, QR code, or existing session.
4.  Adjust notification settings, appearance, and privacy options.
5.  (Optional) Copy the portable folder to a flash drive for mobility.
